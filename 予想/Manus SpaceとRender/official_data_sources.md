# 競艇公式データソース一覧

## 1. ダウンロード可能なデータ（LZH形式）

### 1.1 レーサー期別成績（ファン手帳）
- **URL**: `https://www.boatrace.jp/static_extra/pc_static/download/data/kibetsu/fan{YYMM}.lzh`
- **期間**: 2002年〜現在（年2回更新：前期04、後期10）
- **内容**:
  - 選手基本情報（登番、氏名、支部、級別、年齢、体重など）
  - 全国勝率・複勝率
  - コース別成績（1〜6コース）
  - 平均スタートタイミング
  - 過去の級別推移
- **状態**: ✅ 対応済み（ダウンロード・パース・インポート）

### 1.2 競走成績
- **URL**: `https://www1.mbrace.or.jp/od2/K/{YYYYMM}/k{YYMM}{DD}.lzh`
- **期間**: 2005年1月〜現在（日次更新）
- **内容**:
  - レース結果（着順、タイム、決まり手）
  - 払戻金（3連単、3連複、2連単、2連複、単勝、複勝）
  - 天候・風向・風速・波高
  - スタートタイミング
  - 進入コース
- **状態**: ✅ 対応済み（ダウンロード・パース実装中）

### 1.3 番組表
- **URL**: `https://www1.mbrace.or.jp/od2/B/{YYYYMM}/b{YYMM}{DD}.lzh`
- **期間**: 2005年1月〜現在（日次更新）
- **内容**:
  - 出走メンバー（登番、氏名、級別、支部）
  - 全国勝率・2連率
  - 当地勝率・2連率
  - モーター番号・2連率
  - ボート番号・2連率
  - 今節成績
  - 早見（展示順位）
- **状態**: ✅ 対応済み（ダウンロード・パース実装中）

## 2. リアルタイムデータ（スクレイピング）

### 2.1 オッズ
- **URL**: `https://www.boatrace.jp/owpc/pc/race/odds{TYPE}?rno={R}&jcd={JJ}&hd={YYYYMMDD}`
- **TYPE**: 3t（3連単）, 3f（3連複）, 2tf（2連単/複）, 1t（単勝/複勝）
- **更新頻度**: リアルタイム（締切まで）
- **状態**: ✅ 対応済み（10秒間隔収集、2連単/2連複/単勝/複勝のみ）

### 2.2 直前情報
- **URL**: `https://www.boatrace.jp/owpc/pc/race/beforeinfo?rno={R}&jcd={JJ}&hd={YYYYMMDD}`
- **内容**:
  - 展示タイム
  - チルト角度
  - 部品交換情報
  - 調整情報
  - スタート展示（進入コース、スタートタイミング）
- **状態**: ❌ 未実装

### 2.3 水面状況
- **URL**: 直前情報ページに含まれる
- **内容**:
  - 天候
  - 風向・風速
  - 波高
  - 水温
- **状態**: ❌ 未実装（競走成績には含まれる）

### 2.4 レース結果
- **URL**: `https://www.boatrace.jp/owpc/pc/race/raceresult?rno={R}&jcd={JJ}&hd={YYYYMMDD}`
- **内容**:
  - 着順
  - 決まり手
  - 払戻金
  - レースタイム
- **状態**: ❌ 未実装（競走成績ダウンロードでカバー可能）

## 3. その他のデータ

### 3.1 選手情報
- **URL**: `https://www.boatrace.jp/owpc/pc/data/racersearch/profile?toession={登番}`
- **内容**: 選手プロフィール、通算成績、最近の成績
- **状態**: ❌ 未実装（ファン手帳でカバー可能）

### 3.2 モーター・ボート成績
- **URL**: 各レース場のページ
- **内容**: モーター/ボート別の勝率・2連率
- **状態**: ❌ 未実装（番組表に含まれる）

### 3.3 レース場データ
- **URL**: `https://www.boatrace.jp/owpc/pc/data/stadium?jcd={JJ}`
- **内容**: レース場の特徴、水面状況、コース別データ
- **状態**: ❌ 未実装
