# ボートレース公式サイト URL構造調査

## 基本URL
- ベースURL: `https://www.boatrace.jp/owpc/pc/race/`

## オッズページURL構造

### パラメータ
- `jcd`: 競艇場コード（01-24）
- `hd`: 日付（YYYYMMDD形式）
- `rno`: レース番号（1-12）

### オッズ種別別URL
| 種別 | URL | 説明 |
|---|---|---|
| 3連単 | `/odds3t?rno={rno}&jcd={jcd}&hd={hd}` | 三連単オッズ |
| 3連複 | `/odds3f?rno={rno}&jcd={jcd}&hd={hd}` | 三連複オッズ |
| 2連単・2連複 | `/odds2tf?rno={rno}&jcd={jcd}&hd={hd}` | 二連単・二連複オッズ |
| 拡連複 | `/oddsk?rno={rno}&jcd={jcd}&hd={hd}` | 拡連複オッズ |
| 単勝・複勝 | `/oddstf?rno={rno}&jcd={jcd}&hd={hd}` | 単勝・複勝オッズ |

### その他のページ
| ページ | URL | 説明 |
|---|---|---|
| 出走表 | `/racelist?rno={rno}&jcd={jcd}&hd={hd}` | レース出走表 |
| 直前情報 | `/beforeinfo?rno={rno}&jcd={jcd}&hd={hd}` | 直前情報（天候・展示タイム等） |
| 結果 | `/raceresult?rno={rno}&jcd={jcd}&hd={hd}` | レース結果 |
| レース一覧 | `/raceindex?jcd={jcd}&hd={hd}` | 場別レース一覧 |
| 払戻金一覧 | `/pay?hd={hd}` | 当日払戻金一覧 |

## オッズ更新について
- ページ上に「オッズ更新時間」が表示される（例: 12:09）
- 手動で「更新」ボタンを押すか、ページリロードで最新化
- リアルタイム更新ではなく、リクエスト時点のスナップショット

## 注意事項
- 過去データのダウンロードページは要ログイン
- スクレイピング時はサーバー負荷に配慮が必要
- robots.txtの確認が必要
