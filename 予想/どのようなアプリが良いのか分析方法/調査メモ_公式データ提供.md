# 競艇公式サイト 提供データ調査結果

## 1. 公式ダウンロード可能データ

### 1.1 レーサー期別成績（モーターボートファン手帳）
- **形式**: LZH圧縮ファイル（独自フォーマット、固定長テキスト）
- **提供期間**: 2002年〜2026年（前期/後期で年2回更新）
- **内容**: 選手ごとの詳細な成績データ

| 項目カテゴリ | 含まれるデータ |
|-------------|---------------|
| 基本情報 | 登番、名前、支部、級別、生年月日、性別、身長、体重、血液型 |
| 全体成績 | 勝率、複勝率、1着/2着回数、出走回数、優出/優勝回数、平均ST |
| コース別成績（1〜6コース） | 進入回数、複勝率、平均ST、平均ST順位、着順別回数 |
| 事故情報 | F回数、L0/L1回数、K0/K1回数、S0/S1/S2回数 |
| 期別情報 | 前期/前々期/前々々期の級別、能力指数、養成期 |

### 1.2 競走成績ダウンロード
- **内容**: 全国24場のレース成績
- **注意**: 要ログイン（テレボート会員限定の可能性）
- **形式**: 不明（ログイン後に確認が必要）

### 1.3 番組表ダウンロード
- **内容**: 全国24場の番組表（出走表）
- **注意**: 要ログイン（テレボート会員限定の可能性）

### 1.4 サービス終了情報
> 「番組表閲覧」「オッズ情報・結果」「競走成績閲覧」については、2025年3月5日をもってサービスを終了しました。

## 2. スクレイピングで取得可能なデータ

公式サイトの各ページから取得可能：

| データ種別 | URL形式 | 取得方法 |
|-----------|---------|---------|
| 当日開催場一覧 | `/race/index?hd={YYYYMMDD}` | スクレイピング |
| 出走表 | `/race/racelist?rno={}&jcd={}&hd={}` | スクレイピング |
| オッズ（各種） | `/race/odds3t`, `odds3f`, `odds2tf`, `oddsk`, `oddstf` | スクレイピング |
| 直前情報 | `/race/beforeinfo?rno={}&jcd={}&hd={}` | スクレイピング |
| レース結果 | `/race/raceresult?rno={}&jcd={}&hd={}` | スクレイピング |

## 3. 外部APIサービス

### BoatRace Database API（有料）
- 提供元: team-nave
- URL: https://www.team-nave.com/system/jp/products/brdbapi/
- 機能: CSV形式でのデータダウンロード

### BoatraceOpenAPI（GitHub）
- URL: https://github.com/BoatraceOpenAPI/results
- 形式: JSON
- 内容: レース結果データ（GitHub Pages上で公開）

## 4. 結論

**公式サイトから直接ダウンロードできるデータ:**
- ✅ レーサー期別成績（2002年〜、LZH形式）
- ⚠️ 競走成績・番組表（要ログイン）

**スクレイピングが必要なデータ:**
- オッズ（リアルタイム）
- 直前情報
- レース結果詳細

**公式サイトでは提供されていないデータ:**
- 過去のオッズ時系列データ
- 予想情報
